# 📘 Curso Udemy – Fundamentos de Python

---

## 📖 Sobre o Projeto

Este repositório foi criado com o objetivo de documentar minha evolução no aprendizado da linguagem Python por meio de um curso estruturado na Udemy.

Aqui estão organizados exercícios, implementações práticas e experimentações desenvolvidas ao longo das aulas, com foco na consolidação dos fundamentos da linguagem e no fortalecimento da lógica de programação.

Mais do que um repositório acadêmico, este projeto representa meu compromisso com aprendizado contínuo, disciplina técnica e boas práticas de versionamento.

---

## 🎯 Objetivos de Aprendizado

- Consolidar fundamentos da linguagem Python  
- Desenvolver raciocínio lógico e capacidade de resolução de problemas  
- Aplicar estruturas de dados na prática  
- Organizar código de forma clara e reutilizável  
- Utilizar Git e GitHub como ferramenta de controle de versão  

---

## 🧠 Conteúdos Desenvolvidos

O projeto inclui práticas envolvendo:

- Estruturas condicionais  
- Estruturas de repetição  
- Listas, tuplas, dicionários e conjuntos  
- Funções com parâmetros e retorno  
- Manipulação de coleções  

---

## 🛠 Tecnologias Utilizadas

- Python 3  
- Git  
- GitHub
